## Pub/Sub topics + Pub/Sub subscriptions

```bash
gcloud pubsub topics create myTopic
gcloud  pubsub subscriptions create --topic myTopic mySubscription
```

> Create a Pub/Sub topic.
^
> Create Pub/Sub Subscription.
